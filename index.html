입점제안 메일 생성기
정보 입력 → 제목·본문 각 4가지 선택 → 복사
📋 사용 방법
처음 쓰시는 분도 3분이면 메일 완성! 순서대로 따라해 주세요 😊
1
내 정보 입력
• 이름과 직책은 필수예요
• 직책은 팀장 / 매니저 / 셀렉매니저 등 직책명만 입력
• 카카오ID · 이메일은 선택 — 입력하면 서명에 자동 반영
• 서명 미리보기 확인 후 다음 눌러주세요
2
카테고리 선택
• 제안할 제품이 속하는 카테고리를 눌러주세요
• 선택하면 해당 카테고리 실제 성과 수치가 메일에 자동으로 들어가요
3
제품명 입력
• 제안할 제품 이름 전체를 입력해주세요
• 띄어쓰기 포함 그대로 입력
• 예) 앱솔리 또또빵 제주 감귤 원액
• 전송 버튼 또는 Enter로 다음 단계
4
업체명 입력
• 제안하려는 브랜드·업체 이름 입력
• 모르면 미정 버튼 클릭 → 메일에 '귀사'로 표시
5
제품 특징 입력 (선택)
• 제품 핵심 특징을 한 줄로 간단히 입력
• 예) 날개없는 무소음, 아이방용
• 없으면 건너뛰기 버튼 클릭
6
제목 + 본문 선택 후 복사
• 제목 4가지 중 선택 → 제목 복사
• 본문 4가지 중 선택 → 본문 복사
• 또는 제목+본문 전체 복사로 한 번에!
• 메일 앱에 붙여넣기 후 발송
• ⚠️ 입점제안서 + 판매 레퍼런스 파일 꼭 첨부!
💡 TIP
• B안 제안이유 먼저 / C안 임팩트 먼저 / D안 수치 강조 / E안 짧고 간결
• 처음엔 B안이나 E안이 읽기 편해요
• 발신자 정보는 매번 다시 입력해야 해요 (저장 기능 없음)
• 궁금한 점은 메리에게 문의해주세요 😊
